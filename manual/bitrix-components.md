# Компоненты Битрикс

Компонент Битрикс обычно содержит файл component.php, шаблоны в templates/, class.php для современного подхода и дополнительные файлы result_modifier.php, component_epilog.php.

Современный компонент лучше строить вокруг класса, который наследуется от CBitrixComponent. В class.php обычно размещается executeComponent(), подготовка параметров, загрузка данных и подключение шаблона.

Файл template.php отвечает за вывод HTML. Не стоит размещать тяжелую бизнес-логику прямо в template.php.
